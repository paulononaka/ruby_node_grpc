# Node gRPC Client

> Service 2: Products list

## Requirements

This application runs on [Docker](https://www.docker.com/).

## Setup

```
docker build -t client .
```

# Usage

```
docker run client
```

... or just `rake` if you have Rake and Ruby already installed.
